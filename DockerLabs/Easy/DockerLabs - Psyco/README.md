# DockerLabs - Psyco

La resolución incluye la identificación de vulnerabilidades críticas, como directory path traversal, la obtención de
credenciales sensibles y la escalada de privilegios mediante técnicas avanzadas, como el abuso de binarios
configurados y un ataque de library hijacking. A lo largo del análisis, utilicé herramientas de enumeración,
análisis de configuración de servicios y enfoques creativos para explotar las debilidades del sistema.
Finalmente, logré acceder al sistema con privilegios de superusuario, completando con éxito este desafío
técnico
